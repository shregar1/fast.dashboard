# fastx_dashboards

**Dashboard and analytics UI integrations** for FastMVC applications.

---

## Overview

This package is a **placeholder or thin integration layer** for embedding operational dashboards (metrics, charts, admin views) into FastMVC services. Concrete widgets and framework bindings depend on your front-end stack (React, HTMX, etc.).

Install from the monorepo when your application template references it:

```bash
python -m pip install -e ./fastx_dashboards
```

Check `pyproject.toml` (if present) for the distribution name and dependencies.
